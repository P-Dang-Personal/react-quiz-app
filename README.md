# Video Game Quiz App
This is a quiz app for video games, press 'Start quiz' to begin answering 5 questions about various video games. Uses Open Trivia DB to generate questions / answers to display on the screen

# Technologies Used
- React (useState,useEffect etc.)
- [Open Trivia DB](https://opentdb.com/)
- [html-entites] (https://www.npmjs.com/package/html-entities)
- [clsx] (https://www.npmjs.com/package/clsx)
